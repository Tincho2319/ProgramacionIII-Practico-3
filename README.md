# 🚀 Impulsar - Soluciones Digitales

**Transformamos ideas en soluciones digitales innovadoras**

Sitio web corporativo para una empresa de desarrollo de software que ofrece servicios de desarrollo web, aplicaciones mobile, soluciones cloud y más.

---

## 📋 Descripción del Proyecto

Impulsar es un proyecto académico que consiste en un sitio web profesional para una empresa de software. Incluye páginas de inicio, servicios, equipo, FAQ, contacto y un formulario de solicitud de servicios.

---

## ✨ Características

- ✅ Página de inicio con información general
- ✅ Catálogo de servicios
- ✅ Presentación del equipo
- ✅ Formulario de solicitud de servicios
- ✅ Sección de preguntas frecuentes
- ✅ Formulario de contacto
- ✅ Diseño responsivo

---

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura y semántica
- **CSS3** - Estilos y diseño
- **JavaScript** - Interactividad (próximamente)
